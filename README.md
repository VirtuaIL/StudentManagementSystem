# Student Management API

The primary goal of this repository is to demonstrate and practice the following:
*   **Java 17 & Spring Boot:** Building scalable web services, utilizing Spring IoC/DI, and handling data with the Collections API.
*   **RESTful Design:** Implementing standard HTTP methods (GET, POST) to communicate between client and server.
*   **Dockerization:** Packaging applications into lightweight, portable containers for consistent environments.
*   **CI/CD Fundamentals:** Automating builds and testing using GitHub Actions.
*   **Build Automation:** Managing project lifecycles and dependencies with Maven.

## Tech Stack
*   **Backend:** Java, Spring Boot (Web)
*   **Build Tool:** Maven
*   **Containers:** Docker
*   **Automation:** GitHub Actions (Workflows)
